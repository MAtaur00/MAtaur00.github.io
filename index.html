Personal projects

Collection of games that I've helped to create
as a member of the team

  1. Gun.Smoke, fan made version -  This was the first project I was part of in the university. As a group of 4, we made a version of the old arcade game, Gun.Smoke. I was in charge of the coding. Here's the link to the latest release
  2. Dungeons & Dragons: Wrath of Silumgar - For the second project in university, I joined with 6 friends to make a new, fan made version of Dungeons & Dragons: Shadow over Mystara. In this project, I worked as 2D artist and designer. Here's the link to the latest release
  3. The Witcher: A Bard's Tale - As the third project in university, the whole class worked together to make this fan made The Witcher game. I worked as a member of the design team that created the game. Here's the link to the latest release
  4. ArcAIde - Together with another classmate, we created this resource management game that takes place in an Arcade Saloon. This project was made with Unity. In this project, we both worked as designers and programmers Here's the link to the latest release
  5. Érebos: Worldbuilding and character creation - As a part of my final project in university, I created and designed an imaginary world and some of the characters that live in it. Here's the link to the worldbuilding project and here's the link to the document with the characters
